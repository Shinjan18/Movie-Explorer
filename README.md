# 🎬 Movie Explorer

Movie Explorer is a React-based web application that allows users to explore movies, search for titles, and browse trending, popular, upcoming, and top-rated movies using the TMDB API.

Live Demo  
 https://movie-explorer-three-rho.vercel.app/
 

## 🚀 Features
- 🔎 Search for movies by title
- 🔥 Trending movies (Home page)
- ⭐ Popular movies
- 🎥 Now Playing movies
- 📅 Upcoming movies
- 🏆 Top Rated movies
- 📄 Pagination support
- 🌙 Dark themed UI
- ⚡ Fast performance using Vite


## 🛠️ Tech Stack
- React
- Vite
- JavaScript
- CSS
- TMDB API
- Vercel (Deployment)
- GitHub (Version Control)
